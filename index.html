Custom Currency (CoinsEngine)

Overview

Designed and implemented a custom "Silver" currency system using CoinsEngine. This included full Vault integration, PlaceholderAPI support for dynamic display, and configuration of economy settings for server-wide use.

Key Features

  • Custom Silver currency with ⚡ symbol
  • Vault integration for cross-plugin compatibility
  • PlaceholderAPI support for real-time balance display
  • Configured transfer settings and leaderboards

Proof

Configuration Sample

Name: Silver Symbol: ⚡ Command_Aliases: silver Column_Name: coins Decimal: false Start_Value: 0.0
GUI SYSTEM

Rank Shop Menu (DeluxeMenus)

Overview

Built a complete rank purchase system with DeluxeMenus featuring a custom GUI, Silver currency integration, and automated rank assignment through LuckPerms commands.

Key Features

  • Custom purple/black themed GUI
  • 4 purchasable ranks: Iron, Gold, Diamond, Emerald
  • Currency integration with CoinsEngine
  • Automatic rank assignment via LuckPerms
  • Price-based requirements (1k, 3k, 5k, 10k Silver)

Proof

Configuration Sample

menu_title: '&8[&5Rank Shop&8]' open_command: shop size: 27 items: 'iron_rank': material: IRON_INGOT slot: 10 display_name: '&f&lIron Rank' lore: - '&7Cost: &f1,000 ⚡ Silver'
