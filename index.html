DISCORD BOT

Vanish Discord Bot

Overview

Comprehensive Discord moderation bot built for the Vanish Guild Discord server, this guild was created on the UltimisMC Minecraft server. This bot features advanced permission hierarchies, bilingual English/Arabic support, and sophisticated member management systems.

Key Features

  • Bilingual support (English/Arabic) across all commands
  • Role-based permission system (Leader, Manager, Admin, Officer)
  • Advanced moderation (warn, mute, kick, ban)
  • Temporary role assignment system
  • DM notification systems
  • Clean embed formatting without visible user IDs

Technical Implementation

Built with Discord.js and Node.js, implementing complex command structures with over 10,000 lines of code. Utilizes slash commands, button interactions, and advanced permission checks for secure guild management.

DISCORD BOT MINIGAMES BOT

Vanish Games Discord Bot

Overview

All-in-one Discord gaming bot featuring 11 interactive button-based games with slash commands, cooldown systems, and comprehensive game state management. Built with 1,113 lines of production-ready code.

Available Games

  • Tic Tac Toe - Interactive board game with AI opponent
  • Rock Paper Scissors - Classic choice game
  • Trivia Quiz - Multiple categories (General, Science, History)
  • Math Quiz - Difficulty levels (Easy, Medium, Hard)
  • Dice Roll - Customizable dice games
  • Coin Flip - Heads or tails prediction
  • Magic 8-Ball - Fortune telling responses
  • Higher or Lower - Number prediction game
  • Blackjack - Full casino-style card game
  • Guess the Number - Random number guessing
  • Would You Rather - Community voting game

Technical Implementation

Built with Discord.js featuring slash commands, button interactions, embed builders, and action rows. Implements per-game cooldown systems, active game state tracking, and configurable game settings. Uses Collections for efficient data management and includes admin configuration commands.
