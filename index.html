Minecraft PluginIn Progress

VanishItems v2

Overview

Full rebuild of VanishItems for Vanish SMP. Where v1 had hardcoded ability logic in Java, v2 moves everything to a config-driven registry. New abilities can be added entirely in YAML without touching any code.

Features

  • Fully YAML-driven ability definitions with triggers, targets, cooldowns, and point costs
  • Ability categories: Combat, Utility, Passive, Visual
  • Triggers: ON_HIT, ON_KILL, ON_RIGHT_CLICK, ON_BLOCK_BREAK, PASSIVE, ON_DAMAGED, ON_SNEAK_RIGHT_CLICK
  • Effect stack system for config-driven abilities without custom handlers
  • Shop integration with price points per ability
  • Armor hide via PacketEvents

  • Passive: Armor set bonuses, Speed on Kill, Night Vision, Silent Steps, No Fall Damage
  • Visual: Particle Burst, Ambient Aura

VanishColors v1

Overview

Custom name and chat colour plugin built for Vanish SMP. Players can personalise their name colour and message colour with standard colours, hex codes, gradients, or rainbow. All features are permission-gated via LuckPerms.

Features

  • Name colour and message colour customisation independently
  • Standard colours, hex, gradients (up to 3 stops), and rainbow
  • GUI colour picker accessible via /vcolor
  • PlaceholderAPI expansion for display in tab, scoreboard, etc.
  • SQLite-backed persistent storage
  • Admin commands to change other players colours

Gory Bot

Overview

Full-featured moderation bot built for gvtt3d's Discord server (discord.gg/EFKCNAXdrc). Built on Discord.js v14 with a SQLite database, covering everything from moderation to giveaways to staff tracking. Currently in active development.

Features

  • Full moderation suite: warn, mute, kick, ban, tempban, case management
  • Ticket system with categories, ratings, and transcript logging
  • Appeal system with cooldowns, modal forms, and staff review
  • Giveaway system with required roles, bonus entries, and pausing
  • User report system with cooldowns and staff action buttons
  • Staff activity tracking and logging
  • Message snipe, edit/delete logging, member join/leave logging
  • Per-guild config system, bypass system, and more
